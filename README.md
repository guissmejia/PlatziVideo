# Proyecto-PlatziVideo

This is the creation of the Frontend for the PlatziVideo project using React.js

### [DEMO](https://platfix.herokuapp.com/)

### Install

```sh
npm install
```

### Run Tests

```sh
npm run test
```

### Stack

- React.js
- Redux
- SASS
- Webpack
- Babel
